Sobre mí

Soy un técnico en tecnologías de la información con especialización en infraestructura de redes digitales. Me gusta el mantenimiento de computadoras y brindar soporte técnico. Aunque aún no tengo mucha experiencia, disfruto aprender y me apasiona diseñar sistemas de redes robustas y seguros que optimizan el rendimiento y aseguren la protección de los datos. Estoy comprometido a mejorar continuamente mis habilidades para ofrecer soluciones eficientes y confiables en el ámbito de las TI.

Experiencia

  • Proyecto de control de acceso en la universidad (Estadia)

    Fecha: Mayo 2024 - Agosto 2024

    Desarrollé un sistema de control de acceso para una universidad utilizando procesamiento de imágenes, interfaces amigables para el usuario y algoritmos de seguridad de datos. Implementé bases de datos en Excel y gestión de roles de usuario. Habilidades desarrolladas: Programación, manejo de bases de datos, control de acceso, seguridad informática.

  • Proyecto: Sistema de Registro de Herramientas con Lector QR.

    Fecha: Agosto 2024 - Noviembre 2024

    Desarrollé un sistema de registro y control de herramientas utilizando un lector de códigos QR. El proyecto incluyó la implementación de una base de datos MySQL para el almacenamiento de registros, integrando la captura de datos a través de un sistema web accesible para usuarios. Implementé funciones para la autenticación de usuarios, el manejo de credenciales de acceso y el seguimiento del uso de herramientas mediante códigos QR escaneados. Además, el sistema generaba informes automatizados en formato Excel para el análisis y seguimiento del uso de los equipos.



    Habilidades desarrolladas:
    1. Programación web (PHP, JavaScript)
    2. Manejo de bases de datos MySQL
    3. Implementación de lectores QR y automatización de procesos
    4. Generación de informes automatizados
    5. Seguridad y autenticación de usuarios en sistemas web
